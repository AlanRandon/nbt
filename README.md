# SNBT

## Divergence

- The SNBT implementation does not support unquoted field names beginning with `0-9`, `-`, `.`, or `+`
- The SNBT implementation allows trailing commas
- The SNBT implementation does not allow `uuid` or `bool` to be used as unquoted strings or keys
